Vulnerabilities

This application contains a number of deliberate (and probably also unintended) security vulnerabilities. Some of the example vulnerabilities that you can "exercise" are listed below:

XSS Attack :: Product Search field

The search field on the Product page displays the search keywords entered. This allows you carry out a malicious XSS attack by entering JavaScript code. Here are some examples of what you could enter in the "keywords" field:

<script>alert("I am hacking!")</script>

or:
